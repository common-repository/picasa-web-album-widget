== Description ==

This plugin will make it possible to add random picasa images to your wordpress site.
Just fill in your picasa username and select how many images you want to visualise.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `picasaWebAlbumWidget.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
